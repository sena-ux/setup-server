# Installasi
```
sudo apt install apache2-util
htpasswd -c /etc/nginx/.htpasswd codeserver
```
# Konfigurasi NGINX
```
location / {
    auth_basic "Restricted";
    auth_basic_user_file /etc/nginx/.htpasswd;

    proxy_pass http://127.0.0.1:8080;
}
```

# Update Password
Tanpa menggunakan -c karena -c untuk create user baru
```
sudo htpasswd /etc/nginx/.htpasswd codeserver
```
